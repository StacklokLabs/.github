# .github repository Overview
The .github repository (not to be confused with the .github directory that you may find a repository) is a special repository that is home to global templates that will be available as a fallback for any repository that hasn't defined specific templates in their .github directory. 

Reference: [GitHub documentation](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file)
